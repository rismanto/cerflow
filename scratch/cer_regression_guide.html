Bagian 1 — Langkah-Langkah di R Studio
1
Siapkan Data

Export data dari CERFlow ke format CSV dengan kolom berikut:

siswa18578342

2
Load Data di R
# Load data dari file CSV
data <- read.csv("cerflow_data.csv")

# Cek struktur dan ringkasan data
str(data)
summary(data)
3
Korelasi Sederhana (Baseline)
# Korelasi Pearson: map score vs post-test (tanpa covariate)
cor.test(data$map_score, data$posttest_score, method = "pearson")
Ini adalah baseline — korelasi mentah sebelum mengontrol pengaruh feedback. Catat nilai r dan p-value dari output ini untuk dibandingkan dengan model regresi.
4
Multiple Regression dengan Feedback sebagai Covariate
# Model 1: Sederhana (tanpa covariate)
model1 <- lm(posttest_score ~ map_score, data = data)
summary(model1)

# Model 2: Dengan feedback count sebagai covariate
model2 <- lm(posttest_score ~ map_score + count_feedback, data = data)
summary(model2)

# Model 3: Full model (tambah total_actions)
model3 <- lm(posttest_score ~ map_score + count_feedback + total_actions,
            data = data)
summary(model3)

Bagian 2 — Interpretasi: Kedua Prediktor Signifikan
Pertanyaan kunci: Jika p-value pada map_score dan count_feedback keduanya < 0.05, apakah keduanya sama-sama bisa memprediksi skor post-test?

Jawaban singkat: Ya — tapi dengan cara yang sangat berbeda.

Ketika kedua prediktor signifikan dalam model yang sama, artinya keduanya secara independen menjelaskan variansi unik dalam post-test score, setelah mengontrol satu sama lain. Namun peran dan interpretasinya berbeda:

Aspek map_score count_feedback
Arah koefisien Positif (+0.681) Negatif (−1.203)
Interpretasi Skor peta ↑ → post-test ↑ Feedback lebih banyak → post-test ↓
Peran dalam model Prediktor utama (hipotesis penelitian) Covariate (dikontrol, bukan fokus)
Makna teoretis Pemahaman konseptual Ketergantungan pada bantuan/hints
Ilustrasi penting: Dua siswa bisa mencapai map_score yang sama (misalnya 85), tetapi melalui jalan yang berbeda:
 • Siswa A: skor 85, pakai feedback 1× → memahami materi secara mandiri
 • Siswa B: skor 85, pakai feedback 8× → mengandalkan hints untuk memperbaiki jawaban
Post-test akan "melihat" perbedaan ini — Siswa A cenderung lebih unggul. Koefisien negatif pada count_feedback menangkap sinyal ini.
Temuan tambahan yang berharga: Signifikannya count_feedback menunjukkan bahwa perilaku help-seeking selama rekonstruksi adalah indikator kualitas belajar yang independen — terlepas dari skor peta itu sendiri. Ini bisa menjadi temuan tambahan menarik dalam paper Anda.
Bagian 3 — Formulasi Hipotesis

Berdasarkan model ini, Anda dapat mengajukan dua hipotesis:

HipotesisPernyataanPrediktor

Contoh Kalimat Pelaporan untuk Paper

"A multiple regression analysis was conducted to examine whether CER map score predicted post-test performance after controlling for feedback use during reconstruction. Results indicated that map score was a significant positive predictor of post-test scores (β = 0.58, p < .001). Count of feedback requests was also a significant negative predictor (β = −0.21, p = .005), indicating that students who relied more heavily on feedback hints during reconstruction tended to score lower on the post-test, even when controlling for their final map score. Together, the model accounted for 71% of the variance in post-test performance (R² = .71, F(2, n−3) = ..., p < .001)."

  • → map_score signifikan positif: pemahaman konseptual memprediksi learning outcome
  • → count_feedback signifikan negatif: ketergantungan pada hints mengurangi performa post-test
  • → R² = .71: model menjelaskan 71% variansi skor post-test
